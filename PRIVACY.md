# Privacy Policy

**Lesson Total Package** is a free, open-source plugin for Codex.

## Data Collection

This plugin does not collect, store, or transmit any personal data.

- No user information is gathered.
- No external servers are contacted by this plugin.
- No usage analytics or telemetry are recorded.

## Third-Party Services

This plugin does not integrate with any third-party services or APIs on its own.
All AI processing is handled entirely by Codex and its underlying model provider.

## Changes

If this policy changes, the updated version will be committed to this repository.

## Contact

For questions, open an issue at the repository on GitHub.
