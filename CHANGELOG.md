# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.0] - 2026-04-15

Initial public release of `lesson-total-package`.

### Added

- Codex plugin metadata in `.codex-plugin/plugin.json`
- Core skill package in `skills/class-total-package/`
- Reference guides and validation scenarios for package composition
- `README.md`, `LICENSE`, `PRIVACY.md`, and `TERMS.md`
- Repository `.gitignore`

### Highlights

- Supports selective lesson-package composition instead of forcing a fixed bundle
- Designed to combine board-writing, mind map, PBL, and inquiry-report assessment workflows
- Includes prompt examples and composition rules for Korean classroom planning use cases
